Voici une application de Gestion d'une Collection de cartes Pokémon pour mobiles
sous Android développée sous Android Studio.

Cette application ne contient pour l'instant que la première Génération de pokémon,
le Set de Base du Wizard et les formats "Normal" et "Edition 1".

Ce projet a été réalisé dans le cadre d'études en DUT Informatique mais sera poursuivi par la suite.

Depuis la soutenance, la page de statistiques globales a été poursuivie en ajoutant
des statistiques par Format de cartes et par Type de cartes.

La page de statistiques par Sets est en cours de développement.

La fonctionnalité de rechercher une carte via un nom de pokemon et de filtres
est à prévoir.

L'apk de l'application est disponible dans le dossier app/build/outputs/apk/debug/.
